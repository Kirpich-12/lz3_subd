### **Задания по подзапросам**

1.  Вывести список студентов, средняя оценка которых на **20%** меньше максимальной по университету (подзапрос).
2.  Вывести список студентов, средняя оценка которых равна максимальной по университету (подзапрос).
3.  Вывести список фамилий студентов, которые учатся на самом многочисленном факультете (подзапрос).
4.  Вывести список студентов, для формы, курса, факультета которых число иностранцев равно **0** (подзапрос).
5.  Определить всех сокурсников студента **Богоявленского**, с которыми он учится на одном курсе.
6.  Найти студентов, которые учатся на тех же курсах, что и студент **Зингель** и студентка **Зайцева**.
7.  Вывести список студентов, которые учатся на факультетах, курсах и формах, на которых учится более одного иностранца (без отчества).
8.  Вывести список иностранцев, с указанием общего числа студентов обучающихся с ними на потоке (один факультет, курс).

### **Задание по процедурам**

1.  Создать хранимую процедуру для подсчета количества студентов на факультете и форме обучения (входные данные: название факультета и наименование формы обучения. Результат вывести в процессе выполнения процедуры).
2.  Создать хранимую процедуру для подсчета числа предметов на факультетах (наименования), а также числа одинаковых предметов (без входных данных, результат вывести через выходные переменные отдельно для каждого факультета, использовать их при формировании результата с вида: **Для ФПК читается ___ предметов, для ФПМ читается ___ предметов, всего предметов ___ (из которых идентичных)) (print)**.
3.  Создать хранимую процедуру: добавления студента на указанный факультет и форму обучения (входные данные: название факультета и наименование формы обучения, дата рождения, дата поступления, ФИО). Зачислять на первый курс. Если указанной формы, факультета или их комбинации не существует, то возвращать сообщение об ошибке.

### **Задание по функциям**

1.  Создать функцию скалярную, которая при передаче ей отчества студента, будет возвращать значение **"иностранец"** или **"гражданин"**.
2.  Создать функцию которая будет выводить нагрузку (сумму часов) для каждого преподавателя.

### **Задание по VIEW**

1.  Создайте представление:
    * с информацией ФИО, курс, форма обучения всех студентов ФПК;
    * с информацией Факультет, курс, количество общих часов, в которое войдут курсы заочной формы;
    * с информацией об количестве отличников на каждом курсе-факультете (оценка $>8$);
    * с информацией о слабоуспевающих студентах ($<6$) и их среднем балле.
2.  Определите какие из созданных представлений являются модифицирующими, а какие только чтение и почему.

### **Задания по UNION**

1.  Вывести список преподавателей с указанием надбавки к зп: количество часов более **450** — **20%**, от (**450** до **300**) — **10%**, (**300** и менее) — **0%**.
2.  Вывести список фамилий студентов и преподавателей с указанием гражданства (если есть отчество то **РБ**, если нет — **иностранное**).
3.  Вывести список преподавателей работающих и на **ФПК** и на **ФПМ**.
4.  Вывести список преподавателей **ФПК**, которые не работают на **ФПМ**.
5.  Сформировать запрос, который вернет количество студентов, количество преподавателей и всего человек вида:

| Студентов | [число] |
| :--- | :--- |
| Преподавателей | [число] |
| Всего человек | [число] |
